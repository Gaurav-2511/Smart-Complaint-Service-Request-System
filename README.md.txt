# Smart Complaint & Service Request System

## Project Overview

Smart Complaint & Service Request System is a full-stack web application developed using Spring Boot and Angular.

This system allows users to register, login, submit complaints, and track complaint status. Admin can manage complaints, update status, and monitor complaint statistics through an admin dashboard.
